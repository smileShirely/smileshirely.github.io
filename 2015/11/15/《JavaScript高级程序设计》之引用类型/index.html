《JavaScript高级程序设计》学习笔记

Object类型

就Object在应用程序中的存储和传输数据而言，这种引用类型还是十分理想的选择

创建Object实例的方式有两种

  • 一种是使用new操作符后跟Object构造函数

  • 一种是使用对象字面量表示法

在使用对象字面量语法的时候，属性名也可以是字符串

在实际开发过程中，还是能倾向于对象字面量语法

访问对象属性的方法有两种：

  • 点表示法 如 alert(person.name);
  • 方括号表示法 如 alert(person[“name”]);//将要访问的属性

    Arry类型

    栈方法（后进先出）

push(推入) pop(弹出)

队列方法(先进先出)

结合使用shift()和push()方法，就能实现和队列一样的方式使用数组。
shift()能够使移除数组中的第一项并返回该项，同时使数组长度减一。
另外还有一种unshift()的方法，通过使用unshift()和pop()方法，就可以从相反的方向来模拟队列，即在数组的前端添加项，从数组末端移除项。

重排序的方法

  • reverse()反转数组的顺序

  • 默认情况 sort()方法会按升序排列数组项，为了实现排序，sort()方法会调用每个数组的toString()转型方法，然后比较得到的字符串，以确定如何排序。

因为sort()方法是根据字符串比较的，所以在这种情况下排序就是错误的，可见sort()的排序方法并不是最佳的。

  • 通过比较函数来排序

对于数值类型或者其valueOf()方法返回值类型的对象类型，可以用更简单的比较函数

操作方法

  • concat()方法：在没有给concat()方法传递参数的时候，它只是+ 复制当前数组并返回文本；如果传递是一个或多个数组，则会将每一项都添加到结果数组中去；如果传递的值不是数组，就会被简单的添加到结果数组的末尾。
  • slice()方法：

当slice()方法只有一个参数的情况下，该方法返回从该参数指定位置开始到数组末尾的所有项，有两个参数时，就是确定的位置范围内的所有数组

  • 最强大的就是splice()方法

位置方法

indexOf和lastIndexOf。这两个都有两参数，一个是要查找的项，一个是查找起点位置的索引（可选）。在没有查找到的情况下返回-1

迭代方法

下面几种方法的相同点是：对数组中的每一项运行给定的函数

  • every()：若该函数的每一项都返回true，则返回true
  • filter():返回该函数中会返回true的项构成的数组
  • forEach():没有返回值
  • map()： 返回每次函数调用的结果组成的数组
  • some():任意一项返回true，则返回true

var numbers = [1,2,3,4,5,4,3,2,1];
numbers.forEach(function(item,index,array){
//执行某些操作
});

Date类型

ECMAScript中的Date类型实在早起Java中的java.util.Date类型基础上构建的，Date类型使用自UTC中的标准时间。所以Date类型保存的日期可以精确到1970年1月之前或之后的285 616年。
Date,parse()方法接收一二表示日期字符串参数，然后尝试根据这个字符串返回相应的日期毫秒数

RegExp类型

通过RegExp类型来支持正则表达式
var expression = / pattern /flags;
每个正则表达式都可带一个或多个标志

  • g :表示全局（global）模式，即模式将被应用于所有字符串，而非在发现第一个匹配项时立即停止
  • i:表示不区分大小写（case-insensitive）模式
  • m:表示多行（multiline）模式，即在到达一行本文末尾时还会继续查找下一行中是否存在与模式匹配的项

RegExp的每个实例都具有下列属性，通过这些属性可以取得有关模式的各种信息

  • global:布尔值，表示是否设置了g标志
  • ignoreCase:布尔值，表示是否设置了i标志
  • lastIndex:整数，表示开始搜索下一个匹配项的字符位置，从0算起
  • multiline:布尔值，表示是否设置了m标志
  • sourse: 正则表达式的字符串表达式，按照字面量形式而非传入函数中的字符串模式返回
